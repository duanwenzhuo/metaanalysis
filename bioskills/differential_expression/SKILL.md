# bioskills / differential_expression

Differential expression analysis for single-cell RNA-seq data.

## Skills

### deseq2.py
DESeq2-based differential expression analysis (via rpy2 bridge).
- **bioSkills**: `bio-deseq2`

### edger.py
EdgeR-based analysis for between-group gene expression differences.
- **bioSkills**: `bio-edger`

### de_results.py
Aggregates and filters differential expression results with fold-change and p-value thresholds.
- **bioSkills**: `bio-de-results`
