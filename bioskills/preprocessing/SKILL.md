# bioskills / preprocessing

Data preprocessing and normalization for single-cell RNA-seq.

## Skills

### data_io.py
Read/write single-cell data in multiple formats (10x MTX, h5ad, loom, txt).
- **bioSkills**: `bio-single-cell-data-io`

### hvg.py
Identify highly variable genes (HVG) for dimensionality reduction.
- **bioSkills**: `bio-single-cell-hvg`

### normalize.py
Library size normalization, log-transformation, and scaling.
- **bioSkills**: `bio-single-cell-normalization`

### qc.py
Quality control: filter cells by counts, features, mitochondrial percentage.
- **bioSkills**: `bio-single-cell-qc`

### scale.py
Linear scaling and regression of confounding variables.
- **bioSkills**: `bio-single-cell-scaling`
