# bioskills / enrichment

Gene set enrichment and pathway analysis for single-cell data.

## Skills

### go_enrichment.py
Gene Ontology (GO) enrichment analysis using statistical overrepresentation tests.
- **bioSkills**: `bio-go-enrichment`

### gsea.py
Gene Set Enrichment Analysis (GSEA) for ranked gene lists.
- **bioSkills**: `bio-gsea`

### gsva.py
Gene Set Variation Analysis (GSVA) for pathway activity scoring.
- **bioSkills**: `bio-gsva`
